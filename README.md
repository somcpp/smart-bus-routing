## Setup

1. Create virtual environment (recommended):

```bash
python -m venv venv
source venv/bin/activate  # on Windows: venv\\Scripts\\activate
```

2. Install deps:

```bash
pip install -r requirements.txt
```

3. Run app:

```bash
python app.py
```

4. Open http://127.0.0.1:5000 in your browser
